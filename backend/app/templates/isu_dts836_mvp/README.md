# ISU DTS.836 MVP Şablonu

Bu klasörde OMRChecker şablon dosyaları bulunmalıdır:

- `template.json` — OMRChecker form şablon tanımı (bölge koordinatları, soru grupları)
- `config.json` — OMRChecker işleme parametreleri (eşik değerleri, görüntü boyutu)

Bu dosyalar optik form tasarımına özgüdür ve ayrıca temin edilmelidir.
Faruk, eski sistemdeki bu dosyaları buraya kopyalamalıdır.
